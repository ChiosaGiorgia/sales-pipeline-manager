# sales-pipeline-manager
Sales pipeline management system with CLI - University project
## Description
This project is a command-line tool to manage sales opportunities and leads.
Built as part of Software Engineering course at Ca' Foscari University.

## Features (planned)
- Import/export data via CSV
- SQLite database for data persistence  
- CLI interface with argparse
- Lead and opportunity tracking

## Requirements
- Python 3.12+
- SQLite3
- Standard library only (csv, argparse)
